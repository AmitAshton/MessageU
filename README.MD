\# פרויקט MessageU (תכנות מערכות דפנסיבי)



פרויקט זה מממש מערכת צ'אט מאובטחת מבוססת שרת-לקוח, כחלק מדרישות הקורס "תכנות מערכות דפנסיבי" (20937).



השרת ממומש ב-Python 3 והלקוח ממומש ב-C++11.



---



\## הערה לבודק (רועי רחמני)



\*\*כפי שסוכם במייל, הפרויקט מוגש באיחור באישור עד לתאריך 9.11.25.\*\*



---



\## דרישות סביבה (Prerequisites)



\### צד שרת (Python)



\* \*\*Python 3.x\*\*.

\* אין צורך בספריות חיצוניות (הפרויקט משתמש בספריות מובנות בלבד: `socket`, `sqlite3`, `threading` וכו').



\### צד לקוח (C++)



\* \*\*מערכת הפעלה Windows\*\*.

\* \*\*Visual Studio 2019\*\* (כנדרש ב-PDF של הפרויקט).

\* \*\*ספריית Crypto++:\*\* יש להתקין את הספרייה באמצעות מנהל החבילות `vcpkg`.

&nbsp;   1.  התקן את `vcpkg` (אם לא מותקן).

&nbsp;   2.  הרץ ב-PowerShell: `.\\vcpkg install cryptopp:x64-windows`

&nbsp;   3.  הרץ ב-PowerShell: `.\\vcpkg integrate install` (פעולה זו מקשרת אוטומטית את הספרייה ל-Visual Studio).

\* \*\*Winsock:\*\* הספרייה מקושרת אוטומטית דרך הקוד (`NetworkManager.h`).



---



\## הוראות הרצה



כדי לבדוק את המערכת, יש להריץ את השרת ולאחר מכן להריץ שני לקוחות (או יותר) במקביל.



\### 1. הרצת השרת (Python)



1\.  פתח חלון טרמינל (CMD או PowerShell).

2\.  נווט לתיקיית השרת: `cd src/server`

3\.  צור קובץ `myport.info` המכיל את הפורט הרצוי. לדוגמה:

&nbsp;   ```bash

&nbsp;   echo 1234 > myport.info

&nbsp;   ```

4\.  הפעל את השרת:

&nbsp;   ```bash

&nbsp;   python main.py

&nbsp;   ```

5\.  השרת ירוץ, ייצור אוטומטית קובץ `defensive.db` (בסיס הנתונים) וקובץ `server.log` (לוגים), ויאזין לחיבורים בפורט שהוגדר.



\### 2. קומפילציה והרצת הלקוח (C++)



\#### א. קומפילציה (Build)



1\.  פתח את הקובץ `src/client/client.sln` באמצעות Visual Studio 2019.

2\.  ודא שפלטפורמת היעד (Solution Platform) מוגדרת ל-`x64` (כדי להתאים לספריית Crypto++ שהותקנה).

3\.  בצע Build לפרויקט (`Build > Build Solution` או `F7`).

4\.  פעולה זו תיצור קובץ `client.exe` בתיקייה `src/client/x64/Debug`.



\#### ב. הרצה (Testing)



\*\*חשוב:\*\* כל לקוח חייב לרוץ מתיקייה נפרדת כדי לשמור קובץ `my.info` (זהות) משלו.



1\.  צור תיקייה ייעודית לבדיקות, למשל `C:\\MessageU\_Test`.

2\.  בתוכה, צור שתי תיקיות נפרדות עבור שני לקוחות, למשל `Alice` ו-`Bob`.

3\.  \*\*העתק\*\* את הקובץ `client.exe` (שיצרת בשלב הקומפילציה) לתוך תיקיית `Alice` \*\*וגם\*\* לתוך תיקיית `Bob`.

4\.  \*\*צור קובץ `server.info`\*\* בתוך תיקיית `Alice`. הקובץ צריך להכיל שורה אחת עם ה-IP והפורט של השרת. (אם השרת רץ מקומית, התוכן צריך להיות):

&nbsp;   ```

&nbsp;   127.0.0.1:1234

&nbsp;   ```

5\.  צור קובץ `server.info` זהה גם בתיקיית `Bob`.



\*\*כעת ניתן להתחיל בבדיקה:\*\*



1\.  פתח חלון טרמינל עבור "אליס" ונווט לתיקייה שלה (`cd C:\\MessageU\_Test\\Alice`).

2\.  הרץ `client.exe`.

3\.  פתח חלון טרמינל \*\*נוסף\*\* עבור "בוב" ונווט לתיקייה שלו (`cd C:\\MessageU\_Test\\Bob`).

4\.  הרץ `client.exe`.

5\.  בצע את הפעולות בתפריט (כגון רישום `110` בשני הלקוחות, בקשת רשימת משתמשים `120` וכו').
